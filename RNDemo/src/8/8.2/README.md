# alert / console: 开发周期的调试工具
* alert(): 简单的弹窗提示
* 区别 Alert.alert()，这个是 RN 的对话框工具
* console.log: 最常用的控制台输出
* console 日志输出分级
* console.log 字符串模版和占位符三种方式: %o、%s、%d
* console.log 添加样式: %c 颜色和字号

    * color: orange
    * font-size: x-large、x-medium、x-small

* console.log 输出组件树
* console.log 输出表格日志
* console.log 输出分组日志
